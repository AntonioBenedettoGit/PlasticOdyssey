# Plastic Odyssey 🌊

**Plastic Odyssey** è un'esperienza immersiva di scrollytelling che visualizza l'impatto ambientale dell'inquinamento da plastica. Attraverso un viaggio interattivo ad alta fedeltà, ripercorre il ciclo di vita di una singola bottiglia di plastica: dalla sua creazione industriale alla sua trasformazione finale in microplastiche persistenti nell'oceano.

## ✨ Caratteristiche

- **Scrollytelling Dinamico**: Animazioni fluide legate allo scorrimento che raccontano una storia continua.
- **Visual SVG Morphing**: Una bottiglia di plastica centrale con effetto 3D che si trasforma (ruota, scala e si svuota) man mano che l'utente avanza.
- **Scene Atmosferiche**: Quattro fasi narrative distinte:
    1. **La Genesi**: L'origine industriale e la creazione basata sul petrolio.
    2. **Consumo Rapido**: L'uso effimero di un oggetto monouso.
    3. **L'Abbandono**: Il viaggio dei rifiuti di plastica verso l'oceano.
    4. **Erosione Infinita**: La decomposizione in microplastiche e il loro ingresso nella catena alimentare.
- **UI Immersiva**: Overlay di texture granulose, immagini di sfondo con effetto parallasse e uno sfondo dinamico a gradiente radiale che cambia colore in base al mood della storia.
- **Ottimizzazione delle Performance**: Sviluppato con tecnologie web moderne per garantire uno scorrimento a 60fps e layout reattivi.

## 🛠️ Stack Tecnologico

- **React & TypeScript**: Per un'architettura a componenti robusta e sicura.
- **Vite**: Strumento di build e server di sviluppo ultra-rapido.
- **Tailwind CSS**: Per una tipografia raffinata e layout utility-first.
- **Framer Motion**: Gestisce le trasformazioni avanzate legate allo scroll e le animazioni basate sulla fisica.
- **Lucide React**: Per un'iconografia coerente ed elegante.

## 🚀 Per Iniziare

### Prerequisiti

- Node.js (v18 o superiore consigliata)
- npm o yarn

### Installazione

1. Clona il repository:
   ```bash
   git clone <repository-url>
   ```
2. Installa le dipendenze:
   ```bash
   npm install
   ```
3. Avvia il server di sviluppo:
   ```bash
   npm run dev
   ```

Per creare una build ottimizzata per la produzione:
```bash

```
L'output verrà generato nella cartella `dist/`.

## 🎨 Direzione Creativa

Il progetto utilizza un'estetica "Dark Mode" per sottolineare la gravità dell'argomento trattato. Combina una tipografia audace (Space Grotesk, Archivo Black) con immagini di sfondo cinematografiche e animazioni SVG fluide per mantenere alto il coinvolgimento durante tutto lo scorrimento.

# English Translation 🇬🇧

**Plastic Odyssey** is an immersive scrollytelling experience that visualizes the environmental impact of plastic pollution. Through a high-fidelity interactive journey, it traces the lifecycle of a single plastic bottle—from its industrial creation to its final transformation into persistent microplastics in the ocean.

## ✨ Features

- **Dynamic Scrollytelling**: Smooth, scroll-linked animations that tell a continuous story.
- **Morphing SVG Visuals**: A central 3D-like plastic bottle that transforms (rotates, scales, and empties) as the user progresses.
- **Atmospheric Scenes**: Four distinct narrative phases:
    1. **La Genesi**: The industrial origin and oil-based creation.
    2. **Consumo Rapido**: The ephemeral usage of a single-use object.
    3. **L'Abbandono**: The journey of plastic waste toward the ocean.
    4. **Erosione Infinita**: The decomposition into microplastics and their entry into the food chain.
- **Immersive UI**: Grainy texture overlays, parallax background images, and a dynamic radial gradient background that shifts color with the story's mood.
- **Performance Optimized**: Built with modern web technologies for 60fps scrolling and responsive layouts.

## 🛠️ Technical Stack

- **React & TypeScript**: For a robust, type-safe component architecture.
- **Vite**: Ultra-fast build tool and development server.
- **Tailwind CSS**: For refined typography and utility-first layouts.
- **Framer Motion**: Powering the advanced scroll-linked transforms and physics-based springs.
- **Lucide React**: For consistent, elegant iconography.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create an optimized production build:
```bash
npm run build
```
The output will be generated in the `dist/` directory.

## 🎨 Creative Direction

The project uses a "Dark Mode" aesthetic to highlight the gravity of the subject matter. It combines bold typography (Space Grotesk, Archivo Black) with cinematic background imagery and fluid SVG animations to maintain high engagement throughout the scroll.
